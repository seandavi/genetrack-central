Welcome

GeneTrack is a bioinformatics framework that allows scientists to:

  • Share sequencing data within a lab or organization using a project based access.
  • Automatically run statistical tools to generate quality control results.
  • Visualize genomic data in various contexts via an easy to use track builder.
  • Generate peak calls and predictions for various binding events.
See the Installation page for more details.

GeneTrack demo

See GeneTrack visualizations for CHiP-Seq data:

  • Drosophila H2AZ nucleosome predictions, 454 sequencing

Using with Galaxy

Users may access GeneTrack from within Galaxy. They may visualize their datasets and deposit GeneTrack generated peak predictions and scores into their history. For a demonstration see the Galaxy tutorial
