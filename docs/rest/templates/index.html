Welcome

GeneTrack is a python based bioinformatics framework designed to visualize genomic data. Its main focus is accessiblity while providing sophisticated features for advanced users. We aim to build a simple and flexible software tool that can be deployed with minimal effort withouth sacrificing efficiency and performance.

Get GeneTrack

To accomodate adding new data analysis visualizations GeneTrack is published in frequent (backward compatible) releases. The file naming follows is in the form genetrack-2.0.0-2009-035.zip The first version numbers indicate the release version of GeneTrack (in this case 2.0.0). The second version numbers indicate the year and day of the release 2009-035 means the 35th day of 2009.

  • source code and Windows binaries can be found in downloads
  • git repository git://github.com/ialbert/genetrack.git
