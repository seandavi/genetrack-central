GeneTrack is a Python based genomic data analysis and visualization framework. It aims to provide generic building blocks for creating sophisticated web based genomic data visualization sites.

It runs on multiple platforms such as: Windows, Unix, MacOSX and has modest computational requirements. It provides an embedded database and webserver to allow for a simple and straightforward deployment for small traffic websites. It may also be run behind high performance servers such as Apache.

The site that you are visiting is used to present visualizations, for more details on the system's capabilities and installation see the main GeneTrack site.
