GeneTrack is a Python based genomic data analysis and visualization framework. It aims to provide generic building blocks for creating sophisticated web based genomic data visualization sites.

It runs on multiple platforms such as: Windows, Unix, OSX and has modest computational requirements. It provides an embedded database and a webserver to allow for a simple and straightforward deployment for small traffic websites.

For more details on the system's capabilities and installation see the Main GeneTrack site.
